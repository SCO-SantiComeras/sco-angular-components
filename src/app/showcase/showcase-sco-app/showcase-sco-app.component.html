Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
headerFixed Indicador para añadir a la clase header la propiedad fixed boolean false
translatePipe Indicador para utilizar el pipe de translate en los textos del componente boolean false
logoPath Path del logo a mostrar string './../../../resources/img/sco-logo.png'
logoWidth Anchura del logo number 80
logoHeight Altura del logo number 40
maxMenuItems Máximo de elementos del menú, si se sobrepasa el límite se mostrara el menu mobil number 6
menuItems Lista de elementos a mostrar en el menú MenuItem []
menuBorderRadius Indicador que marca el tipo de borde del menú boolean true
contentPadding Padding del contenedor 'content' number 20
contentBackgroundColor Color del contenedor 'content' string undefined
toastEnabled Indicador para habilitar los toast boolean true
toastOrientation Orientación por donde aparecerá el toast string 'top-right'
  • ScoToastConstants.ORIENTATION_TOP_RIGHT
  • ScoToastConstants.ORIENTATION_TOP_LEFT
  • ScoToastConstants.ORIENTATION_BOTTOM_RIGHT
  • ScoToastConstants.ORIENTATION_BOTTOM_LEFT
toastBlock Indicador para mostrar el toast expandido en modo bloque boolean false
spinnerEnabled Indicador para habilitar el spinner boolean true
spinnerEmbedded Indicador que el spinner contenga la clase embedded boolean false
spinnerImgPath Path de la imagen a mostrar en el spinner string './../../../resources/img/spinner.gif'
menuMobileTransparent Indicador para modificar el background del botón del menu mobil boolean true
menuMobileIcon Clase de Font Awesome para proveer el icono del botón del menu mobil string 'fa fa-bars'
menuMobileColorRevert Indicador para revertir el color del botón del menu mobil (#FFF) boolean true
menuMobileShowIcons Indicador para mostrar los iconos de los elementos del menú mobil boolean true
menuMobileHeaderMargin Margen superior de la cabecera del menú mobil number 60
Nombre Descripción Valor devuelto
itemClick Evento que devuelve el componente al seleccionar un elemento del menú MenuItem
