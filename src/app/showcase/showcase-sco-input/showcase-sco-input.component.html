    onInputKeyUp($event: string) {

  • ScoInputTypeConstants.NUMBER
  • ScoInputTypeConstants.EMAIL
label Texto de la etiqueta del input string undefined
showLabel Indica si se muestra la etiqueta del input boolean false
icon Clase de boostrap para mostrar en el icono string undefined
showIcon Indica si se muestra o no el icono del input boolean false
enabledIconClick Indica si está hábilitado el evento click del icono boolean false
readonly Indica si el input esta en modo solo lectura boolean false
disabled Indica si el input esta deshabilitado boolean false
placeholder Texto a mostrar en el input mientras no contenga valor string ''
inline Indica si el input esta en modo en linea (Label - Input) boolean false
required Indica si el input es obligatorio boolean false

iconClick Evento al pulsar el icono del input boolean

inputKeyUp Evento al levantar la tecla del input string
