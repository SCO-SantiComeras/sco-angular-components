Nombre Descripcción Tipo Valor por defecto Posibles valores
menuItems Lista de elementos a mostrar en el menú MenuItem []
menuButtonTransparent Indicador que modifica el background del botón del menú boolean true
menuButtonIcon Clase de font awesome para proveer el icono string 'fa fa-bars'
menuButtonIconColorRevert Indicador para modificar el color del icono del botón del menú (#FFF) boolean false
showMenuIcons Indicador para mostrar los iconos de los elementos del menú boolean true
translatePipe Indicador para utilizar el pipe de translate en los textos del menú boolean false
headerMargin Margen superior de la cabecera number 0
Nombre Descripcción Valor devuelto
itemClick Evento que devuelve el componente al seleccionar un elemento del menú MenuItem
