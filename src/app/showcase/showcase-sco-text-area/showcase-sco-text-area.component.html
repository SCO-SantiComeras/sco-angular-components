Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
placeholder Texto cuando el input esta vacio string ''
label Label asociado al input string undefined
name name del textarea string undefined
disabled Deshabilita el textarea boolean false
readonly Indica si esta en modo consulta boolean false
maxlength Máxima longitud del textarea number undefined
minlength Mínima longitud del textarea number undefined
required Indica si es obligatorio o no boolean false
validate Permite la validación del textarea boolean false
inline Coloca los elementos del textarea en linea boolean false
rows Número de filas del textarea number 4
cols Número de columnas del textarea number 10
